# Netlify-Discord-Integration
A file for use in Netlify Functions that interacts with a webhook.
To use this, replace the webhook URL and the image URL in the `discord-webhook.js` file with your own links. After that, put the file into a folder(either a custom one, or the default, /netlify/functions/). In order for it to actually work, you must deploy from a repository. 
Do whatever you want with this.
